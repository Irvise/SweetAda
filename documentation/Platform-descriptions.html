14 Altera10M50GHRD

Altera10M50GHRD is a virtual platform in SweetAda, usable by means of the QEMU emulator. This platform is based on a softcore Nios II processor, implemented in real world on an FPGA.

QEMU instantiates a few peripherals for this platform: - 2 timers - 16550-style UART

By using libioemu, 8 32-bit I/O ports are available.

CPU parameters are fixed at some particular addresses, as dictated by QEMU, and already wrapped in the platform’s configuration.in.
